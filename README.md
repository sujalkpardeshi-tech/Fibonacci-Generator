# Fibonacci Generator (Python)

Overview
This project implements a Fibonacci sequence generator using Python generators. 
It efficiently produces Fibonacci numbers using `yield`, making it memory-efficient.

 Features
- Generate Fibonacci series
- Uses generator function
- Memory efficient
- Supports infinite sequence

Run
python fibonacci_generator.py
